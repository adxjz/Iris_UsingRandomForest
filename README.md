# 🌸 Iris Flower Classification Using Random Forest

## 📌 Overview

This project implements the **Random Forest Classifier** to classify iris flowers into different species using the famous **Iris Dataset** from Scikit-learn.

The model learns patterns from flower measurements and predicts the species of a flower based on its sepal and petal dimensions.

---

## 🎯 Objectives

* Understand the Random Forest Machine Learning algorithm.
* Perform data preprocessing and analysis.
* Train and evaluate a classification model.
* Save the trained model using Pickle.
* Gain practical experience with Scikit-learn.

---

## 🛠️ Technologies Used

* Pandas

---

## 📊 Dataset Information

The Iris dataset contains 150 samples of iris flowers with the following features:

| Feature      | Description          |
| ------------ | -------------------- |
| Sepal Length | Length of sepal (cm) |
| Sepal Width  | Width of sepal (cm)  |
| Petal Length | Length of petal (cm) |
| Petal Width  | Width of petal (cm)  |

### Target Classes

* Iris Setosa
* Iris Versicolor
* Iris Virginica

---

## ⚙️ Model Workflow

1. Load the Iris Dataset
2. Split data into training and testing sets
3. Train the Random Forest Classifier
4. Evaluate model performance
5. Predict flower species
6. Save the trained model as `.pkl`
